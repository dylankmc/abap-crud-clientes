# CRUD com Relatório ALV (ABAP)

Projeto desenvolvido em ABAP utilizando Eclipse (IDE) com intuito de treinar ABAP clássico com ALV e Dynpro.

## Funcionalidades
- Cadastro de dados (Create)
- Leitura e listagem de registros (Read)
- Atualização de informações (Update)
- Exclusão de registros (Delete)
- Relatório utilizando ALV (ABAP List Viewer)

## Tecnologias
- ABAP
- Eclipse (IDE)
- ALV (REUSE_ALV_GRID_DISPLAY)
- Dynpro

## Desafios 
- Entendimento da estrutura de tabelas internas
- Implementação do ALV
- Organização da lógica do CRUD

## Como funciona

O relatório principal exibe um ALV com dados da tabela ZCLIENTES.

Através de botões, o usuário pode acessar as transações:
- Criar (Z001)
- Editar (Z002)
- Deletar (Z003)
- Ler (Z004)
